Strong work experience in micro-service architecture as part of Nokia's upcoming project Nokia Digital Automation Cloud. Thereby, I’m involved mainly in the following microservices:

Private Network Management.

Java-based micro-service, which involves private network slicing like creation, deletion of the private network, resource allocation, deallocation, network reset, network shutdown and subscription management

  • Technologies involved: Java, gRPC, Maven, Cassandra, Docker, Prometheus.

Hardware Inventory Management nad Network Hardware Manager.

Go lang based micro-service. which acts as a Hardware Inventory, responsible for keep tracking of all Hardware's, addition of hardware's , deletion of hardware's , maintaining the hardware activation status.

Network Hardware Group manager which is responsible for the network hardware group, deletion of the network group, mapping and demapping of the hardware to the network group.

  • Technologies involved: go-lang, docker, gRPC, dep, Cassandra.

Configuration Management Service

Go lang based micro-service, which is responsible for maintaining all type network configurations of the router, switch, up-link, Access point, Quality of Service for the given network group.

  • Technologies involved: go-lang, docker, gRPC, dep, Cassandra.
